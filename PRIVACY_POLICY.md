# 개인정보처리방침 (SMU eCampus Dashboard)

시행일: 2026-03-03

SMU eCampus Dashboard(이하 "확장프로그램")는 사용자 데이터를 다음과 같이 처리합니다.

## 1. 수집하는 정보

확장프로그램은 다음 정보를 처리할 수 있습니다.

- 웹사이트 콘텐츠: `https://ecampus.smu.ac.kr/*` 페이지의 과목/학습 항목 정보(과제, 강의, 토론, 자료, 공지의 제목, 링크, 상태, 일정 등)
- 로컬 설정 정보: 필터, 숨김 항목, 패널 상태 등 사용자 UI 설정
- 로컬 캐시 정보: 최근 불러온 과목/항목 데이터

## 2. 이용 목적

- 대시보드 UI 표시 및 기능 제공
- 사용자 설정 유지
- 사용자 요청 시 파일 다운로드 기능 제공

## 3. 저장 위치 및 보관 기간

- 데이터는 사용자 브라우저의 `chrome.storage.local`에 저장됩니다.
- 확장프로그램 삭제 또는 사용자가 브라우저 저장 데이터를 삭제할 때까지 보관됩니다.

## 4. 제3자 제공 및 외부 전송

- 확장프로그램은 수집/처리한 데이터를 개발자 서버 또는 제3자 서버로 전송하지 않습니다.
- 제3자에게 사용자 데이터를 판매하거나 공유하지 않습니다.

## 5. 권한 사용

- `storage`: 설정 및 캐시 저장
- `tabs`: 팝업에서 eCampus 탭 열기/이동
- `downloads`: 사용자 클릭 시 자료/영상 다운로드
- host permissions (`https://ecampus.smu.ac.kr/*`): 해당 사이트에서 기능 제공을 위한 페이지 내용 접근

## 6. 이용자 권리

사용자는 언제든지 확장프로그램 삭제 또는 브라우저 저장 데이터 삭제를 통해 데이터 처리를 중단하거나 삭제할 수 있습니다.
